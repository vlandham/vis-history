
# VIS HIStory

Dashboard for showing a visual history of static sties hosted in git repos.

# Building

Install dependencies:

```
npm install
```

Build files:

```
npm run build
```

Or watch and serve files:

```
npm run watch
```

Then navigate to [http://localhost:8080/](http://localhost:8080/).


# Uses:

* ESLint for Linting
* Webpack for Building
* Babel for ES6
* React for UI
